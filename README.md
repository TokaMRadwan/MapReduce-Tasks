# Hadoop MapReduce Tasks

This repository contains Hadoop Streaming MapReduce tasks implemented using Python Mapper and Reducer scripts. The projects demonstrate data processing, aggregation, and statistical analysis using Hadoop and Python.

## Technologies Used

* Hadoop HDFS
* Hadoop Streaming
* Python
* Linux (CentOS)

## Tasks

### 1. Apache Log Analysis

Analyzes Apache web server logs and counts the number of requests generated by each IP address.

**Output Example**

```text
100.2.4.116    6
100.43.83.137  84
106.78.19.160  18
```

### 2. Course Data Analysis

Processes course records and calculates statistical measures for each city.

The reducer computes:

* Number of records (Count)
* Average score (Mean)
* Standard Deviation (Std)

**Output Example**

```text
Beijing   count=4   mean=97.0   std=17.83
Shanghai  count=4   mean=82.0   std=25.82
```

### 3. Fruits Frequency Analysis

Counts the occurrence of each fruit in the dataset.

**Output Example**

```text
Apple      2
Banana     2
Guava      3
Watermelon 3
```

### 4. Shakespeare Word Count

Implements the classic Word Count problem using Shakespeare's text dataset.

**Output Example**

```text
"A"       4
"Amen"    2
"God"     2
"King"    1
```
